# ☕ Bear Cafe Discord Bot

บอท Discord ประจำร้าน Bear Cafe ที่ถูกพัฒนาขึ้นเพื่อดูแลระบบคะแนนจากการคุยเสียง (Voice Points) และระบบสุ่มแชทหาเพื่อนแบบส่วนตัว (Secret Chat)

## ✨ ฟีเจอร์หลัก (Features)

### 🎙️ ระบบนับเวลาห้องเสียง (Voice Points System)
* ตรวจจับการเข้า, ออก, และย้ายห้องเสียงของสมาชิกในเซิร์ฟเวอร์
* คำนวณระยะเวลา (วินาที) และจำนวนคนที่อยู่ในห้อง เพื่อส่งไปคำนวณแต้มผ่าน API (`VOICE_POINTS_URL`)
* ส่งข้อมูลอัปเดตสถานะการเข้าออกห้องไปที่ Webhook (`WEBHOOK_URL`)
* มีระบบ Heartbeat ช่วยรีเฟรชข้อมูลสถานะเสียงทุกๆ 15 นาที เพื่อป้องกันข้อมูลตกหล่น
* ระบบจะข้ามการแจกคะแนนหากผู้ใช้อยู่ในหมวดหมู่ที่ถูกยกเว้น (Category ID: `1145057060686397611`)
* มีระบบ Health Check ผ่าน HTTP Port (ค่าเริ่มต้น 8000)

### 💬 ระบบสุ่มโต๊ะลับ (Secret Chat Matchmaking)
* **หน้าร้าน (Lobby):** แอดมินสามารถพิมพ์คำสั่ง `b!reset-match` เพื่อสร้างข้อความ Embed พร้อมปุ่มสำหรับกดเข้าคิวสุ่มแชท
* **ระบบเข้าคิว (Queue):** เมื่อมีผู้ใช้งานกดปุ่ม ระบบจะนำเข้าคิว และเมื่อมีครบ 2 คน ระบบจะทำการจับคู่ให้ทันที
* **การสร้างห้องส่วนตัว:** บอทจะสร้าง Text Channel ใหม่ในหมวดหมู่ลับ (Category ID: `1494308739220770888`)
* **ความปลอดภัยขั้นสุด:** ห้องที่สร้างจะปิดไม่ให้คนอื่นเห็น (`@everyone: ViewChannel false`) และเปิดสิทธิ์ให้อ่าน/พิมพ์/ส่งรูป เฉพาะผู้เล่น 2 คนที่จับคู่กันเท่านั้น
* **ทำความสะอาดอัตโนมัติ:** เมื่อมีคนกดปุ่ม "🚪 ลุกจากโต๊ะ" บอทจะทำการลบห้องนั้นทิ้งอย่างถาวรเพื่อล้างข้อมูลและรักษาความเป็นส่วนตัว

## 🛠️ ข้อมูลทางเทคนิค (Tech Stack)
* [cite_start]**Runtime:** Node.js เวอร์ชัน 20 (รองรับการรันผ่าน Docker Image `node:20-slim`) [cite: 1]
* **Main Library:** `discord.js` เวอร์ชัน ^14.14.1
* **Dependencies:** `axios` (^1.6.0) และ `dotenv` (^16.4.0)

## ⚙️ การตั้งค่าตัวแปรสภาพแวดล้อม (Environment Variables)
โปรเจกต์นี้ต้องการตัวแปร `.env` ดังต่อไปนี้ในการทำงาน:
* `BOT_TOKEN`: โทเคนของ Discord Bot
* `WEBHOOK_URL`: ลิงก์ Webhook สำหรับส่งข้อมูลสถานะการใช้งาน
* `VOICE_POINTS_URL`: ลิงก์ API สำหรับส่งข้อมูลระยะเวลาเพื่อบันทึกแต้ม
* `PORT`: พอร์ตสำหรับรัน Health Check Server (หากไม่กำหนดจะใช้พอร์ต `8000`)
